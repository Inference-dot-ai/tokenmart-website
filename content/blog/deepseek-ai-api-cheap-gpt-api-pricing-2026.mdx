## When this is the worse deal

**You need a single signed BAA / DPA with one provider.** TokenMart routes across providers; if compliance requires a contract with the model vendor specifically, buy the key direct and skip routing.

**Sub-100-token requests at p95 < 200 ms.** TokenMart's gateway adds a small routing hop (5–10% routing fee, single-digit ms in the median). For sub-200 ms hard targets, a provider-pinned deployment in the same region wins on latency.

**Your monthly spend is under ~$50 and stays there.** The 300M-token signup bonus covers small projects for weeks, but if you never grow past it the routing optimisations do not pay off in absolute dollars.
